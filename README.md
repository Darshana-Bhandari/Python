# Python Basics Practice

A collection of Python programs created while learning the fundamentals of Python programming.

## Topics Covered

### Variables and Data Types

* String
* Integer
* Float
* Boolean
* None Type

### Operators

* Arithmetic Operators
* Relational Operators
* Assignment Operators
* Logical Operators

### Type Conversion

* Converting strings to integers
* Working with different data types

### User Input

* Taking input from users
* Converting input values into appropriate data types

### Practice Programs

* Sum of two numbers
* Area of a square
* Average of two numbers
* Comparison of two numbers

## Learning Objectives

This repository was created to practice and understand:

* Python syntax
* Variables and data types
* Basic operators
* Input and output operations
* Type conversion
* Problem-solving with Python

## How to Run

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Navigate to the project folder:

```bash
cd python-basics-practice
```

3. Run the program:

```bash
python python_basics_practice.py
```
